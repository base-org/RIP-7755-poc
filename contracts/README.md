# RRC-7755 Contracts

This repo consists of work-in-progress implementations of the smart contracts making up [RRC-7755](https://github.com/ethereum/RIPs/blob/master/RIPS/rip-7755.md);

## Getting Started

### Install dependencies

```bash
forge install
```

### Compile contracts

```bash
forge build
```

### Run tests

```bash
make test
```

### Check coverage report

```bash
make coverage
```
